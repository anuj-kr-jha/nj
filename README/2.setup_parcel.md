# Parcel

```bash
bun i -D parcel
```

```bash
bunx parcel index.html
```

## role of parcel

- different dev and prod builds
- Local server
- HMR = Hot Module Replacement
- Auto refresh
- Caching - Fast build
- Image Optimization
- Minification
- Bundling
- Compression
- Consistent Hashing
- Code Splitting
- Differential Bundling (to support legacy browsers)
- Diagnostic
- Pretty Error Reporting
- HTTPS
- Tree shaking (dead code elimination)
